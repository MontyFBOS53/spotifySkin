##Installation
Clone the repository, move the Spotify folder to your Skins folder.
(Documents/Rainmeter/Skins/)

This skin is dependant on the ![SpotifyPlugin.dll](http://rainmeter.net/forum/viewtopic.php?f=18&t=17077).
